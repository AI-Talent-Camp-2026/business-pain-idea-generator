## Экран: Результаты прогона (список идей)

**Контекст:** Прогон завершён.  
**Цель пользователя:** Быстро просмотреть 10–20 идей, открыть детали, выбрать 1–2 и экспортировать.
**Day‑1 MVP:** без дашборда/аккаунтов/избранного/сравнения; фильтры — опционально, не блокер.

### Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Чекбоксы идей | Выбор 1–2 идей (или больше) для экспорта |
| 2 | (Опц.) сортировка | Ускоряет просмотр, но не блокирует Day‑1 |
| 3 | Экспорт | Открывает модал “Экспорт (Markdown)” (копировать/скачать) |
| 4 | “Сгенерировать ещё” | Запускает новый прогон (с тем же фокусом или случайно) |
| 5 | “Сигналы” в строке | В строке показываем только паттерны; ссылки/цитаты — внутри идеи по “Подробнее” |

### Состояния

- **Empty state:** нет результатов (ошибка/прогон не завершён) — показать причину + CTA “повторить”/“новый прогон”.
- **Loading:** скелетоны строк списка.

### Переходы

- Клик по строке идеи → Экран “Детали идеи”
- “Экспорт” → Модал “Экспорт (Day‑1)”
